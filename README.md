# 🎬 Aytek Şayan Dijital Arşiv & İnteraktif Fan Platformu

Aytek Şayan'ın tiyatro, sinema ve dizi kariyerini uçtan uca dijitalleştiren; dinamik veri yapısıyla çalışan, interaktif oyunlarla zenginleştirilmiş kapsamlı ve modern bir fan arşivi projesi.

## 📌 Proje Hakkında
Bu proje, yalnızca statik bir biyografi sitesi değil; kullanıcıların içeriklerle etkileşime girebileceği, yüksek çözünürlüklü medya dosyalarını indirebileceği ve mini oyunlar oynayabileceği tek sayfa (SPA - Single Page Application) mimarisiyle inşa edilmiş bir arşiv sistemidir.

## 🚀 Temel Özellikler

### 📂 Medya ve Veri Deposu
*   **Dinamik Medya Galerisi:** X (Twitter) karakter kesitleri, 9:16 duvar kağıtları, 3:1 header tasarımları, kamera arkası videoları ve sticker paketleri.
*   **Tam Ekran Lightbox:** Fotoğrafların orijinal boyutunda incelenebildiği ve cihaza indirilebildiği özel galeri modülü.
*   **Tiyatro Envanteri:** 2014'ten günümüze kadar tüm sahne performanslarının afiş, özet, tür ve sahne detaylarıyla listelendiği kronolojik yapı.

### 🎮 İnteraktif Modüller (Gamification)
*   **Şerif Furtuna ile Adam Asmaca:** Karakterin ikonik replikleriyle entegre edilmiş, dinamik hata payı hesaplamalı kelime tahmin oyunu.
*   **Bilgi Testi (Quiz) & Kara Bulmaca:** Oyuncunun projeleri ve hayatı hakkında interaktif bulmacalar.
*   **Günün Repliği & Çuval Terapisi:** Kullanıcı etkileşimini artıran günlük modüller.

### ⏱️ Dinamik Bileşenler
*   **Sayaç Sistemleri:** Canlı olarak çalışan Doğum Günü sayacı ve Yeni Bölüme Kalan Süre sayacı.
*   **Haber Bülteni:** Manuel güncellenen güncel duyuru ve haber panosu.

## 🛠️ Kullanılan Teknolojiler

*   **Geliştirme Ortamı:** React.js / Vite
*   **Yönlendirme:** React Router DOM (Kesintisiz sayfa geçişleri)
*   **Veri Yönetimi:** Local JSON ve JS modülleri (Bileşen tabanlı dinamik veri çekimi)
*   **Tasarım & Stil:** Özel CSS mimarisi, CSS Modülleri, Grid/Flexbox sistemleri ve özel animasyonlar (animate-fade vb.).

## ⚙️ Kurulum ve Çalıştırma

Projeyi kendi bilgisayarınızda çalıştırmak için aşağıdaki adımları izleyebilirsiniz:

1. Depoyu klonlayın:

   git clone [https://github.com/kullaniciadiniz/aytek-sayan-arsiv.git](https://github.com/kullaniciadiniz/aytek-sayan-arsiv.git)

2. Proje dizinine gidin:

   cd aytek-sayan-arsiv

3. Gerekli bağımlılıkları yükleyin:

   npm install

4. Geliştirme sunucusunu başlatın:

   npm run dev
